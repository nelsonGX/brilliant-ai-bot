# 遊戲機制調整
## 音效優化
當 <span class="label label-info">凋零怪</span> 被召喚時，僅有附近 <span class="label label-warning">50 格</span> 近的玩家才能聽到；
當 <span class="label label-info">終界龍</span> 被擊殺時，僅有附近 <span class="label label-warning">50 格</span> 近的玩家才能聽到；
當 <span class="label label-info">玩家死亡／達成成就</span> 時，僅有 <span class="label label-warning">死亡者／成就完成者</span> 才能看到其訊息。
