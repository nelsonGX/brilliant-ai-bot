# 遊戲機制調整
## 探險家地圖限制
目前當玩家使用 <span class="label label-info">探險家地圖</span> 後，由於會導致 <span class="label label-danger">伺服器崩潰</span>（具體資訊可參考[官方漏洞報告](https://bugs.mojang.com/browse/MC-249136)），
故本服暫時不開放玩家使用 <span class="label label-info">探險家地圖</span> ，
{% note warning %}
而該錯誤已於 1.20.5 版本修復，
但輝煌遊戲機制仍在 1.20.2 ，故該錯誤還存在。
{% endnote %}
