# 遊戲機制調整
## 銅塊氧還
水瓶 <span class="label label-danger">左鍵</span>  -> 一階一階的 <span class="label label-danger">氧化</span>
水瓶 <span class="label label-success">右鍵</span>  -> 一階一階的 <span class="label label-success">還原</span>

水桶 <span class="label label-danger">左鍵</span>  -> 直接 <span class="label label-danger">氧化</span>
水桶 <span class="label label-success">右鍵</span>  -> 直接 <span class="label label-success">還原</span>

而 <span class="label label-info">已上蠟</span> 的銅系列方塊無法透過此方式
操作 <span class="label label-danger">氧化</span> 與 <span class="label label-success">還原</span> 的程度！
