# 遊戲機制調整
## 合成配方
本服有一些自定義的合成配方 ...

### 晶淵球

使用 <span class="label label-info">晶淵球</span> 丟向除了 <span class="label label-danger">終界龍、凋零怪、玩家</span> 以外的所有生物／怪物，
並於被砸位置掉落 <span class="label label-success">該生物的頭顱</span>，
玩家可以拾取 <span class="label label-success">該頭顱</span>，放於背包、箱子等任意容器，
而當玩家想要召出生物／怪物時，只需將 <span class="label label-success">該頭顱</span> 放置於地上，即可召出。

{% note danger %}
請注意，不建議捕捉帶有 NBT 標籤之生物，
因而造成的生物異常／損失，概不補償！
{% endnote %}
{% note danger %}
請注意，不建議將被晶淵球砸暈的生物，
在召出時，放置於「空間過小」的位置，

雖然我們有防範了一些情況，
但依然有可能發生放出後，生物消失的問題（因為空間不足），
還請玩家特別注意！一樣發生該情況時，不予補償處理。
{% endnote %}

#### 領地權限相關
與 <span class="label label-info">領地權限</span> 集成，倘若玩家沒有領地的 <span class="label label-danger">擊殺動物（animakilling）</span> 權限時，
玩家無法使用 <span class="label label-info">晶淵球</span>  砸暈動物／怪物。

欲在 <span class="label label-info">領地內</span> 放出動物／怪物時，請確認是否已開放以下領地權限：

1. 動物：
    - 生成動物的總權限（animals）
    - 玩家放置生成動物（canimals）
2. 怪物：
    - 生成怪物的總權限（monsters）
    - 玩家放置生成怪物（cmonsters）

因未設置好 <span class="label label-danger">領地權限</span> 導致放出後的動物／怪物遺失情況，不予補償處理。

### 原礦燒磚與還原
<span class="label label-info">銅／鐵／金原礦方塊</span> 可以於熔爐燒成，
<span class="label label-success">銅／鐵／金磚</span> 。

<span class="label label-success">銅／鐵／金磚</span> 可以於工作檯中還原成，
<span class="label label-info">銅／鐵／金原礦</span> 。
<span class="label label-success">銅／鐵／金磚</span> 必須放置於工作檯 <span class="label label-warning">正中間</span> ，
否則無法正確還原為 <span class="label label-info">銅／鐵／金原礦</span> 。

### 冰塊還原

<span class="label label-success">藍冰／冰磚</span> 可以於工作檯中還原成，
<span class="label label-info">冰磚／冰</span> 。
<span class="label label-success">藍冰／冰磚</span> 必須放置於工作檯 <span class="label label-warning">正中間</span> ，
否則無法正確還原為 <span class="label label-info">冰磚／冰</span> 。
