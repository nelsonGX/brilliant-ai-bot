# 遊戲機制調整
## 基岩穿透
當玩家 <span class="label label-danger">右鍵</span> <span class="label label-info">地獄</span> 上層的 <span class="label label-info">基岩</span> 時，
將傳送至 <span class="label label-danger">基岩上方</span> ；
當玩家 <span class="label label-success">左鍵</span> <span class="label label-info">地獄</span> 上層的 <span class="label label-info">基岩</span> 時，
將傳送至 <span class="label label-success">基岩下方</span> 。
