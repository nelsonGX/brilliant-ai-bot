# 其他便捷功能與遊戲玩法
## 電梯建造

你可以在本伺服器內輕鬆的建造一個以上的電梯，
你需要以以下方式排列，不需輸入指令即可建造電梯：

### 石英方塊 + 紅石方塊

           06-     石英方塊
           05-     紅石方塊
           04-     ( 空氣 )
           03-     ( 空氣 )
           02-     石英方塊
           01-     紅石方塊

當然，他是可以繼續往上或往下延伸的，
兩個樓層之間可相鄰的 <span class="label label-warning">最大距離</span> 沒有限制，但至少要相鄰 <span class="label label-danger">2 格空氣</span> 。

### 任意地毯 + 石英方塊

           06-     任意地毯（含覆地苔蘚）
           05-     石英方塊
           04-     ( 空氣 )
           03-     ( 空氣 )
           02-     任意地毯（含覆地苔蘚）
           01-     石英方塊

當然，他是可以繼續往上或往下延伸的，
兩個樓層之間可相鄰的 <span class="label label-warning">最大距離</span> 沒有限制，但至少要相鄰 <span class="label label-danger">2 格空氣</span> 。

但須注意，地毯電梯在建造時， <span class="label label-warning">不同顏色的地毯不能混和搭配</span> ，
例如一樓是 <span class="label label-danger">紅色地毯</span> ，二樓是 <span class="label label-success">綠色地毯</span> 的話，無法正常連通，
須把二樓改成 <span class="label label-danger">紅色地毯</span> 才可連通樓層。

### 任意壓力板 + 石英方塊

           06-     任意壓力板
           05-     石英方塊
           04-     ( 空氣 )
           03-     ( 空氣 )
           02-     任意壓力板
           01-     石英方塊

當然，他是可以繼續往上或往下延伸的，
兩個樓層之間可相鄰的 <span class="label label-warning">最大距離</span> 沒有限制，但至少要相鄰 <span class="label label-danger">2 格空氣</span> 。

但須注意，壓力板電梯在建造時， <span class="label label-warning">不同材質的壓力板不能混和搭配</span> ，
例如一樓是 <span class="label label-danger">石製壓力板</span> ，二樓是 <span class="label label-success">橡木壓力板</span> 的話，無法正常連通，
須把二樓改成 <span class="label label-danger">石製壓力板</span> 才可連通樓層。

### 電梯操作方式
站在電梯上 <span class="label label-success">跳躍</span> 即可上升；
反之 <span class="label label-danger">蹲下</span> 即可下降。
