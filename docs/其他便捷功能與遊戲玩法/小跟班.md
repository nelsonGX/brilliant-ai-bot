# 其他便捷功能與遊戲玩法
## 小跟班
一個會一直跟著你的小跟班，雖然沒特殊技能，但可以一直陪伴你。
透過 <span class="label label-info">/bpet</span>指令，來開啟 <span class="label label-info">小跟班選單</span> ，
在選單中，玩家可以調整是否召喚出小跟班 ... 等。

並且當你隱身時，小跟班也會跟著隱身，
並且 <span class="label label-success">一、二、四分流共通</span> ，你走到哪，小跟班就跟到哪。（除非你隱藏他）
