# 與眾不同的羽森分流（三服）
## 表情符號

在本服，您可以使用除了 <span class="label label-danger">動態表情符號</span> 以外，所有在 [伺服器 Discord 群組](https://discord.brilliantw.net/) 的 <span class="label label-success">靜態表情符號</span> 。
輸入以下指令即可查看全部可用的表情符號列表，點擊想用的表情符號後，即會自動填入至遊戲聊天欄。

<table>
  <tr>
    <th>指令</th>
    <th>功能</th>
  </tr>
  <tr>
    <td>/emojis <頁碼></td>
    <td>查看全部可用的表情符號列表。</td>
  </tr>
</table>
